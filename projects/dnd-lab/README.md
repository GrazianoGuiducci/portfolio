# D-ND Lab

## Summary

D-ND Lab is an autonomous multi-domain research runtime. It runs cycles, writes
reports, attacks claims with a counter-pole, stores falsified paths in a
cimitero, exposes the current field in a dashboard, and promotes stable outputs
into reusable tools or lab templates.

## Problem

Long AI-assisted work often fails because the system forgets prior decisions,
repeats bad hypotheses after context loss, and produces fluent reports without
testing them. D-ND Lab treats research as a loop: propose, falsify, remember,
adjust, and expose state.

## System

- Domain-independent engine: `D-ND_LAB`.
- Domain configurations: physics, finance, Bitcoin regime, research radar, and
  future custom labs.
- Dashboard: public read surface for field state, reports, products, filters,
  trajectory, and operational status.
- SSP path: Scoperte -> Soluzioni -> Prodotti.
