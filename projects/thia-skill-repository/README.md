# THIA Skill Repository

## Summary

The THIA Skill Repository is an archived snapshot of cognitive, operative, and
bridge skills used to structure AI agent behavior across chat, coding, and
server environments.

## Problem

Agent systems become fragile when every behavior is implicit in one prompt. The
skill repository separates thinking modes, operational actions, bridge
functions, conductors, and retired capabilities.

## System

The inspected material uses a three-plane taxonomy:

- Plane 1: cognitive faculties, portable across LLM hosts.
- Plane 2: operative faculties tied to THIA infrastructure.
- Plane 3: bridge faculties connecting environments.
- Conductors: routing layers for different execution contexts.
