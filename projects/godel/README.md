# Godel

## Summary

Godel is the semantic/logical question layer: a tool for inversion, clarification
and structured interrogation of assumptions.

## Portfolio Role

Godel shows how questions become operational objects. A user brings a tension;
the system inverts the point of view, exposes a hidden assumption, captures the
residue, and opens the next useful question.

It is a practical decision operator connected to the wider THIA/D-ND operating
surface.

## What It Shows

- Public/private lens separation: visitors receive short practical output,
  internal nodes can use richer context.
- Tension-first decision support for strategy, research, product and personal
  decision points.
- Memory of residues and follow-up tensions.
- A path from formal inversion logic into a usable public tool.
