# KPhi1-EN

## Summary

KPhi1-EN is a frozen legacy reference for an autopoietic cognitive kernel for
LLM hosts. It packages identity, axioms, guard rules, skills, and evolution
protocols as a portable seed operating system.

## Problem

LLM workflows need portable structure that preserves identity, validation,
memory behavior, and tool-generation patterns across hosts.

## System

The inspected README describes:

- a seed operating system for LLMs;
- axioms P0-P6;
- a VRA cycle for planning, execution, and self-correction;
- modular skills;
- supervisors for coherence and output density;
- a context protocol and skill genesis layer.

## Status

Legacy/frozen reference. Useful in the portfolio as part of the trajectory from
