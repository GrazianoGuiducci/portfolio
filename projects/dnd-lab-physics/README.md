# D-ND Physics Lab

## Summary

D-ND Physics Lab is the first applied domain built for the D-ND Lab engine. The
engine is generic; this repository provides the physics/math domain content,
claims, context, cimitero, and experiment tools.

## Problem

A reusable research engine needs proof that it can leave the original abstract
model and operate on a domain with real claims, failed paths, and generated
experiments.

## System

The inspected repository separates:

- `domain/` for assertions, seed tensions, context, config and cimitero;
- `tools/` for experiment scripts generated by cycles;
- D-ND Lab as the external engine that runs the cycles.

Topics include prime gap structure, spectral signatures, Markov memory layers,
GUE-Poisson crossover, and hidden sequential information.
