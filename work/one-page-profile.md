# Graziano Guiducci — AI Systems / Agentic UX

## Profile

Since 2022 I have been developing self-maintaining agentic systems for complex AI-assisted work: memory, reentry, state surfaces, verification loops, falsification gates, decision operators, Lab cycles and interfaces for reviewable autonomy.

The current work is organized around:

- **AI Lab / D-ND Lab** — self-maintaining research cycles that reconstruct a live field, generate experiments, falsify outputs and update the next move.
- **THIA** — coordination/runtime layer for sites, repositories, Labs, coders, assistants, memory and routing.
- **d-nd-seed** — portable procedures for AI coders: reentry, memory, hooks, safety, skills and Lab patterns.
- **D-ND kernel** — logical-operational source underneath the system, introduced after the operating evidence is visible.

## Transferable value

I build methods and tools for making AI systems more continuous, inspectable, falsifiable and able to preserve intent across long workflows.
